## 🎯 功能说明

### 用户流程

**首次访问：**
1. 打开网站 → 显示登录/注册页面
2. 注册账号（用户名 + 密码）
3. 登录成功 → 引导填写个人简介
4. 创建个人卡片（介绍自己）
5. 进入首页，可以浏览其他用户卡片

**再次访问：**
1. 打开网站 → 自动登录（session）
2. 直接进入首页

### 技术实现

**前端：**
- 单页应用（SPA）架构
- 页面切换使用JavaScript
- 卡片滑动使用CSS Transform
- 留言使用模态框

**后端：**
- Express + MongoDB
- Session认证
- RESTful API
- Socket.IO（保留聊天功能）

## 🚀 部署步骤

### 本地测试

1. 安装MongoDB（见DATABASE-SETUP.md）
2. 安装依赖：
   ```bash
   npm install
   ```
3. 创建.env文件：
   ```env
   MONGODB_URI=mongodb://localhost:27017/anonymous-chat
   SESSION_SECRET=your-secret-key
   PORT=3000
   ```
4. 重命名server-new.js为server.js
5. 启动服务器：
   ```bash
   npm start
   ```

### Railway部署

1. 在Railway项目中添加MongoDB数据库
2. 添加环境变量 `SESSION_SECRET`
3. 推送代码到GitHub
4. Railway自动部署

## ⚠️ 重要提示

1. **数据迁移**：新系统需要用户注册，旧的匿名聊天数据不会保留
2. **兼容性**：保留了原有的随机匹配聊天功能
3. **安全性**：密码使用bcrypt加密，session使用MongoDB存储
4. **扩展性**：可以轻松添加更多功能（好友系统、私信等）
